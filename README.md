# Superstore Sales Data Analysis

# Problem Statement
Analyze business performance using the Superstore Sales dataset.

# Dataset
Superstore Sales Dataset - sourced from GitHub

# Approach
- Analyzed monthly revenue trends by year
- Identified top 10 best selling products
- Analyzed sales by category, region and segment
- Found top 10 states by total sales

# Results
Technology is the top category, West region leads in sales,
California is the top state, and sales peak in Q4 every year
